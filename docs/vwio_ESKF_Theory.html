  In the traditional Extended Kalman Filter (EKF), the observation equation is linearized, the Jacobian matrix of the observation equation with respect to the state variable is found, and then the Kalman filter is updated.

  In the Error State Kalman Filter (ESKF), an estimate of the nominal state x\mathbf{x}x and an estimate of the error state δx\delta \mathbf{x}δx are available. To update the error state, the Jacobian matrix HHH of the observation equation with respect to the error state is calculated.
